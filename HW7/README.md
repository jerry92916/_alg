我選**老鼠走迷宮**的問題，透過深度優先搜尋與回溯法，讓老鼠從指定起點走出迷宮。  
[程式碼](https://github.com/jerry92916/_alg/blob/main/HW7/pathFinder.py)

---

**概念**

*：牆壁（不可通行）

' '：可走的道路

.：目前嘗試中的路徑

+：已確認為死路的路徑  

---

**流程**

1.從起點開始，檢查有沒有超出邊界或撞牆

2.將目前位置標記為走過（.）

3.依序往 右、下、左、上 四個方向遞迴探索

4.若走到迷宮邊界，視為成功找到出口

5.若四個方向都失敗，回溯並標記為死路（+）
